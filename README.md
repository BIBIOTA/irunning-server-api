功能:
<br />
1.與運動社群應用程式Strava串接及第三方登入，取得個人的跑步紀錄
<br />
2.與中央氣象局、政府開放平台API串接，建立資料庫並取得台灣各地及目前所在地(GPS Location)的即時天氣資訊及空氣品質
<br />
3.透過Node.js爬蟲，取得全國路跑賽事資訊並建立資料庫、搜尋賽事
